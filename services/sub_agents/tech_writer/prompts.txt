```python
def calculate_backoff(retry_count: int, base_delay: int = 1) -> int:
    """
    Calculate the exponential backoff delay for network retries.

    This function computes a delay interval that doubles with each subsequent 
    retry attempt, which helps prevent overwhelming a server during outages.

    Args:
        retry_count (int): The number of times the request has already been retried.
        base_delay (int, optional): The initial delay in seconds. Defaults to 1.

    Returns:
        int: The calculated delay time in seconds before the next retry attempt.
    """
    return base_delay * (2 ** retry_count)
```
